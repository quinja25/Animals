### 최종 등급 구조

최종 등급 = **육질등급(1++, 1+, 1, 2, 3)** + **육량등급(A, B, C)**

- **육량등급**: 도체로부터 얻을 수 있는 살코기의 양을 등급화 (도체중량, 등지방두께, 배최장근단면적으로 산출)
- **육질등급**: 근내지방도와 육질의 품질을 종합적으로 판정 (근내지방도, 육색, 지방색, 조직감, 성숙도)

### 육질등급 판정 기준

각 항목중 가장 낮은 등급을 **최저 등급**으로 적용한 뒤, 성숙도를 반영하여 **최종 등급** 결정.

| 항목 | 1++ | 1+ | 1 | 2 | 3 |
|------|-----|-----|---|---|---|
| 근내지방도 | No.7⁺ ~ No.9⁺⁺ | No.5⁺⁺ ~ No.7⁰ | No.4⁰ ~ No.5⁺ | No.2⁰ ~ No.3⁺⁺ | No.1⁰ ~ No.1⁺⁺ |
| 육색 | No.3 ~ No.5 | No.2, No.6 | No.1 | No.7 | 그 외 |
| 지방색 | No.1 ~ No.4 | No.5 | No.6 | No.7 | 그 외 |
| 조직감 | No.1 | No.2 | No.3 | No.4 | No.5 |

**성숙도 반영:**
- 성숙도 1 → **상**, 2~7 → **점**, 8~9 → **결격**
- 성숙도 8~9인 경우 최저등급에서 1개 등급 하락

**예외 규칙:**
- 육색과 지방색이 모두 No.7에 해당하여 2등급인 경우 → 육질등급을 **3등급**으로 판정
- 근내지방도와 육색·지방색·조직감의 평가결과가 2개 등급 이상 차이나는 경우 → 성숙도를 적용하지 않고 최저 등급을 최종 등급으로 함

### 조직감 구분 기준

| 번호 | 구분 기준 |
|------|----------|
| 1 | 탄력성과 지방의 결이 매우 좋으며 수분이 알맞게 침출되고 결이 매우 곱고 섬세하며 고기의 광택이 매우 좋은 것 |
| 2 | 탄력성과 지방의 결이 좋으며 수분이 다소 알맞게 침출되고 결이 곱고 고기의 광택이 좋은 것 |
| 3 | 탄력성과 지방의 결이 보통이며 수분의 침출정도가 약간 많거나 적고 결과 고기의 광택이 보통인 것 |
| 4 | 탄력성과 지방의 결이 포장에 비해 좋지 않은 수준이며 고기의 표면이 건조하거나 수분이 결이 침출되고 결과 광택이 보통에 비해 좋지 않은 것 |
| 5 | 탄력성과 지방의 결이 나쁘며 고기의 표면이 매우 건조하거나 수분이 아주 많이 침출되고 결과 광택이 매우 좋지 않은 것 |

## 모델 검증 방법 및 지표

자동 검증 시스템이 없으므로, 아래와 같이 자체 검증 체계를 구성합니다.

### 검증 전략

1. **학습/검증/테스트 분할**: 시간 기반 분할 (예: 2023~2024 학습, 2025 테스트) → 데이터 누수 방지
2. **Stratified K-Fold 교차검증**: 등급 불균형을 고려한 층화 추출 (K=5)
3. **성별별 검증**: 암/수/거세 각각의 성능을 별도 보고

### 분류 평가 지표

| 지표 | 산식 | 용도 |
|------|------|------|
| **Macro F1-Score** | 클래스별 F1의 산술 평균 | 주 지표 — 등급 불균형 고려 |
| **Weighted F1-Score** | 클래스별 F1의 가중 평균 (샘플 수 기준) | 보조 지표 — 전체 성능 |
| **Top-1 Accuracy** | 정확히 맞춘 비율 | 기본 성능 |
| **Top-2 Accuracy** | 상위 2개 예측 중 정답 포함 비율 | 인접 등급 허용 성능 |
| **Confusion Matrix** | 클래스별 예측 vs 실제 | 오분류 패턴 분석 |
| **Cohen's Kappa** | 우연 일치를 보정한 일치도 | 순서형 분류 신뢰도 |

### 등급 체계별 분리 평가

최종 등급(16개 클래스) 직접 예측이 어려울 수 있으므로 분리 평가:

- **육질등급** (1++, 1+, 1, 2, 3) → 5개 클래스 분류
- **육량등급** (A, B, C) → 3개 클래스 분류
- **최종 등급** = 육질등급 + 육량등급 조합

## 모델 구축 아이디어

### 접근 방식 1: 육질등급 + 육량등급 분리 예측

최종 등급을 직접 예측(16클래스)하기보다, 육질등급(5클래스)과 육량등급(3클래스)을 **각각 예측 후 결합**하는 방식.

- 육량등급은 산식(WINDEX)이 알려져 있으므로 높은 정확도 기대
- 육질등급이 핵심 난이도 → 근내지방도가 가장 중요한 피처

### 접근 방식 2: 기상 피처 엔지니어링

기상 데이터를 사육 기간에 맞춰 집계하여 피처 생성:

| 피처 | 설명 |
|------|------|
| 사육기간 평균/최고/최저 기온 | 전체 사육 기간 기상 통계 |
| 계절별 기상 통계 | 봄/여름/가을/겨울 분리 집계 |
| 폭염·한파 일수 | 극한 기상 노출 횟수 |
| 도축 전 N일 기상 | 도축 직전 스트레스 영향 |
| 일교차 평균 | 기온 변동성 |
| 강수 집중도 | 습도·강수 스트레스 |

### 접근 방식 3: 농장·혈통 피처 활용

| 피처 | 출처 | 설명 |
|------|------|------|
| 두당 면적 | area | 축사 면적 / 사육두수 → 사육 밀도 |
| 사육두수 변화율 | area | 연도별 증감률 → 농장 경영 상태 |
| 농장 폐사율 | death | 농장별 폐사 건수/사육두수 → 농장 관리 수준 |
| 아비별 평균 등급 | lineage + train | 아비 개체의 자손 등급 통계 → 유전적 영향 |
| KPN별 등급 통계 | lineage + train | 종축별 성과 |

### 모델 후보

| 모델 | 장점 | 적합 이유 |
|------|------|----------|
| **LightGBM** | 빠른 학습, 범주형 변수 자체 처리 | 대용량(240만 행) 처리에 적합, 기본 성능 우수 |
| **XGBoost** | 안정적 성능, 과적합 제어 | 하이퍼파라미터 튜닝 후 최종 모델 후보 |
| **CatBoost** | 범주형 변수 강점, 순서형 타겟 지원 | 등급(순서형) 특성에 적합 |
| **Stacking Ensemble** | 다양한 모델 결합 | 최종 성능 향상용 |

### 파이프라인 설계

```
1. 전처리
   ├── 결측값(-99) 처리
   ├── 날짜 파싱 (ABATT_DATE, BIRTH_YMD)
   └── 기상 데이터 병합 (stn + 사육기간)

2. 피처 엔지니어링
   ├── 기상: 사육기간 통계, 계절별, 극한 기상
   ├── 농장: 두당면적, 폐사율, 사육두수 변화
   └── 혈통: 아비·KPN별 등급 통계 (target encoding)

3. 모델링
   ├── 육질등급 모델 (5클래스)
   ├── 육량등급 모델 (3클래스)
   └── 최종 등급 = 육질 + 육량 결합

4. 검증
   ├── 시간 기반 분할 검증
   ├── Stratified 5-Fold CV
   └── Macro F1, Confusion Matrix, Kappa 보고
```
